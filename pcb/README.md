# ScreenX PCB

**WARNING: This PCB is untested. Use at your own risk.**
As many of you may know, shipping times for PCB's and components is killer, it may be a while before this is fully finished.

## Overview
This is the PCB design for the ScreenX project. The board is currently in development and has not undergone testing.
This is what allows the ESP32 to be powered off the ES Lite wheel's internal power, removing the need for a battery or cable

## Safety Notice
Do not use this PCB in any critical applications until it has been fully tested and validated. Assembly and use of untested hardware may result in damage to components or equipment.
