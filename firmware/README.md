# ScreenX ESP32 Firmware

This folder contains the firmware for the ESP32 used in ScreenX.

## Building from Source

### Prerequisites
- Arduino IDE installed
- ESP32 board support added to Arduino IDE
- The required libraries

### Building

1. Clone or download this repository
2. Open the `.ino` file in Arduino IDE
3. Select your ESP32 board: **Tools** → **Board** → **ESP32**
4. Select the correct port: **Tools** → **Port**
5. Click **Upload** to build and flash the firmware
6. Or you could click **Sketch**, then **Export Compiled Binary** to create a .bin file to use inside the companion app.
